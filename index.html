One-time setup

To run on your phone independently, the app needs your own Anthropic API key. You only enter this once — it's saved on your device.

1
Go to console.anthropic.com and sign up free
2
Click API Keys in the left menu → Create Key
3
Copy the key (starts with sk-ant-...) and paste it below
4
Add a payment method — costs only a few cents per search (Claude API pay-as-you-go)

Anthropic API Key

Your key is stored only on this device in your browser's local storage.
It is never sent anywhere except directly to Anthropic.
